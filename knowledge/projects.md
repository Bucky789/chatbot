

## Projects

### Chess Web Application

-I built a chess web application that allows users to play chess in the browser.
The application supports human vs human and human vs AI gameplay.
- Custom chess UI built with React
- Legal move validation
- Check, checkmate, and stalemate detection
- Pawn promotion support
- AI opponent using a lightweight chess engine

## Tech Stack
- React for frontend
- JavaScript for game logic
- FastAPI for backend APIs
- Stockfish-based lightweight AI integration

This project helped me understand game logic, state management, and AI integration.
